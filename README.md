# Expense Tracker

A personal fullstack learning project for tracking purchases and visualizing spending statistics.

The goal of this project is to improve my skills in frontend and backend web development using Angular, Node.js, and PostgreSQL.

---

## Features

- Create, edit and delete purchases with individual items
- Categorize expenses
- View spending statistics
- Monthly and yearly summaries
- Responsive user interface
- REST API backend
- PostgreSQL database integration

---

## Tech Stack

### Frontend
- Angular
- TypeScript
- Angular Material

### Backend
- Node.js
- Express.js
- TypeScript

### Database
- PostgreSQL
- Prisma
